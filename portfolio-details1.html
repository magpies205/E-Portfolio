Web Design and Programming

For this assignment I was tasked to create a webpage that was for a kids camp. I needed to make up a schedule and add a description. I also needed to include some information about the instructors for this summer camp. The main thing I learned from this assignment was how to properly set up tables in an efficient way that is pleasing to the eye. For the schedule part of this assignment it needed to be in a table and needed to be set up nicely. This assignment was very useful becuase now I will be able to look back on it when I need to create another table.
