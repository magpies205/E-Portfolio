Geolocation App Prototype: GoPlay

This project was my senior capstone. I worked with two other developers in my class. We worked with a client from the Montana Repretory Theatre. He wanted us to create a geolocation app for him that allowed the user to open the app in their location and see all of the businesses around them. There would be different colored pins: one for the business information and another for special events the business is putting on. There would be a homepage, notification page, and a map page. The notification page would show notifications of discount codes for these businesses that would only popup when the user is within a certain distance. We created requirement specifications, did many surveys to get user feedback, conducted 7 interviews to gain feedback before we began the designs, created low-fidelity prototypes, created a Swimlane diagram, created an ER diagram to represent the database, and created user scenarios to conduct 10 user tests. We gained lots of good feedback from all of these things. Unfortunately, we were also supposed to work on an existing app to make it look like the new prototypes we created, but this app was too outdated that it was deprecated. This is because it had been created in React about four years before we started working on it. Regardless of this, we still created many high-fidelity prototypes that we were able to gain feedback on and conduct UI testing with. This project gave me lots of experience with working with my team and communicating with our client to create something that was up to his standards.
